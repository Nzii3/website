Configs v2

Ever wanted to have Modmail v3's configuration system in your normal bot? Then this is the migration for you!

Commands​

UsageDescriptionExample
!configView all the configurations in a paginated session
!config listView all the configurations in a compact message
!config set <key> <value>Set a configuration to the according value!config set main_color #5865f2

Default Configurations​

The following configurations are used globally throughout extensions and other systems in your bot.

log_channel - The channel where bot logs and update logs are sent by default
prefix - The prefix of the bot
main_color - This is the main color for the entire bot
success_color - This is the color when anything is successful
error_color - This is the color when anything goes wrong, unsuccessful commands, or a stern warning
error_emoji - Emoji that indicates something was unsuccessful
success_emoji - Emoji that indicates success
vNziie--10/29/2023
!config set success_color #2ecc71

Success
