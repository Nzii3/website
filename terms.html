Terms of Use

Basic Rules​

These rules should be followed regardless of what service your are using (e.g. custom bots, source code usage, etc.)
1. You, your server, and your bot should be following Discord's Terms of Service, Discord's Community Guidelines, and Discord's Developer Policy at all times.

2. Features/source code requested by you can be denied at any time without reason.

3. You must remain in the development server (Nziie Development) for your bot to be online. If you leave the server at any time your bot will go offline until you join back with no refund given.

4. Please do not rush me, I may cancel your services without a full refund if you rush me too much. You can be patient like every other client. ## Refunds1. If you request a refund for your bot, it can be denied at any time without reason.

2. Refunds may never be given under any circumstance due to hosting costs real money and hosting can't be "refunded."

Source Code Guidelines​

Source code requests can be denied at any time without reason; source code for modmail bots and music bots won't be given

⭐ 1. You MUST keep credit in the code in some commands/features in the source code you are given. If you remove ANY credit whatsoever you will no longer be able to recieve source code and will be removed from the development server with any services purchased being canceled. The source code you are given may be connected to an API and can detect if credit was removed, and if credit is removed the bot will not start or the feature(s) won't work.

2. If you leave the server at any time you will recieve the consequences of rule 1
